Sweet & Salty Game

To play this game, You would required to enter 3 numbers, the starting, the end number and numbers per line. The program runs through the range between the start and the end numbers. Whenever the program encounters a number divisible by 3, it prints sweet, and when it encounters a number divisible by 5 it prints salty, and if it's divisible by both 3 and 5, it prints sweet'nSalty. Otherwise it prints the number. However there are rules to enter number which player needs to follows. The range between both numbers must be no less than 200, and no greater than 10000. And the words per line should be at least 1/100th of the range.
